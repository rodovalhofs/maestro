# maestro-skills

CLI de instalação interativa do [Maestro](https://github.com/rodovalhofs/maestro).

```bash
npx maestro-skills setup
```

[![npm](https://img.shields.io/npm/v/maestro-skills)](https://www.npmjs.com/package/maestro-skills)

## Agentes

| Agente | Flag |
|--------|------|
| Cursor | `--cursor` |
| Claude Code | `--claude` |
| Codex | `--codex` |
| Universal | `--universal` |
